Architecture

بنية إضافة، ليس نسخة

مخلب is a plugin, not a fork. It extends OpenClaw without modifying its core.

┌──────────────────────────────────────────────────────────────────┐
│                        OpenClaw                                │
│            (Your existing installation)                       │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ~/.openclaw/                                                   │
│  ├── openclaw.json         ← plugin registry                   │
│  ├── extensions/                                                │
│  │   ├── mkhlab-core/        ← 3 hooks (greeting, arabizi, hijri)│
│  │   ├── whatsapp-channel/   ← WhatsApp Business API            │
│  │   └── telegram-channel/   ← Telegram Bot API                │
│  ├── skills/                                                    │
│  │   ├── prayer-times/SKILL.md                                   │
│  │   ├── translate/SKILL.md                                      │
│  │   ├── saudi-einvoice/SKILL.md                                │
│  │   └── ... (60 skills)                                        │
│  ├── agents/                                                    │
│  │   ├── scholar.md          ← Islamic research agent           │
│  │   ├── translator.md       ← Translation agent                │
│  │   ├── developer.md        ← Arabic coding agent              │
│  │   └── entrepreneur.md     ← Saudi business agent             │
│  └── SOUL.md                ← Arabic persona & behavior        │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
Plugin, not fork. مخلب follows the same pattern as openclaw-china. Drop it into your OpenClaw directory, register the extension, and all 60 skills are immediately available.

Quick Start

التثبيت في 3 خطوات

Clone, configure, and start using Arabic skills in under a minute.

1 استنسخ المستودع 
git clone https://github.com/mousaabumazin/mkhlab.git
cd mkhlab
2 سجّل الإضافة 
// Add to ~/.openclaw/openclaw.json
{
  "mcpServers": {
    "mkhlab": {
      "command": "node",
      "args": ["PATH_TO/mkhlab/extensions/mkhlab-core/index.js"]
    }
  }
}
3 ابدأ الاستخدام 
openclaw
# Now try: /prayer-times or /translate or /saudi-einvoice
