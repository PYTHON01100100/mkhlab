## 1. STC (Saudi Telecom Company)

### STC Developer Portal
- **URL**: https://developers.stc.com.sa/
- **What it does**: Carrier-grade telecom APIs for messaging, authentication, payments, and IoT
- **API Products**:
  - **SMS Messaging API** -- Send, receive, and track SMS to local/international numbers
  - **OTP API** -- Secure one-time password generation and verification
  - **Identity API** -- Frictionless digital identity verification and onboarding
  - **Payment API** -- One-click billing via STC for digital content/services
  - **IoT API** -- SIM connectivity, data usage, and session info for IoT devices on STC network
  - **Network Events API** -- Subscribe to mobile network events (roaming, SIM swap, cancellation) for authorized STC numbers
- **Docs**: Full REST API docs at developers.stc.com.sa with Swagger/OpenAPI specs
- **Pricing**: Free trial with sandbox access; paid plans for production (contact for pricing)
- **Access**: Sign up, create an app in "My Apps", get sandbox credentials, test, then request production approval

### STC Cloud
- **URL**: https://cloud.stc.com.sa/
- **What it does**: Full cloud IaaS/PaaS/SaaS marketplace
- **Services**: Virtual Data Center, ready-made deployments, subscription services
- **Subsidiary**: sccc by stc -- 100+ cloud products across IaaS, PaaS, SaaS
- **Pricing**: Varies by service; marketplace model
- **Access**: Register at cloud.stc.com.sa

## 6. SDAIA (Saudi Data & AI Authority)

### ALLaM -- Arabic Large Language Model
- **What it does**: Arabic-first large language model, built by SDAIA/NCAI with IBM
- **Model Versions**:
  - **ALLaM-2-7B** -- Available on Azure AI Studio and Hugging Face
  - **ALLaM 34B** -- Powers HUMAIN Chat (launched August 2025)
  - **ALLaM 70B** -- Built but not yet publicly available
- **API Access**:
  - **Azure AI Studio**: Deploy ALLaM-2-7B, fine-tune, and use via Azure API. Pay-per-token pricing through Azure
  - **IBM Watsonx**: Access via watsonx.ai studio. Plans from $0 (Lite) to $1,050/mo (Standard)
  - **Hugging Face**: Open-source download of ALLaM 7B for self-hosting
  - **Ollama**: Community-quantized versions available (e.g., ALLaM:7b-q5_K_M)
- **Docs**: Azure AI Model Catalog; IBM Watsonx docs; Hugging Face model card
- **Training Data**: 500+ billion Arabic tokens (largest Arabic dataset ever assembled for ALLaM 34B)

- **URL**: HUMAIN Chat app (iOS, Android, web)
- **What it does**: Arabic-first conversational AI powered by ALLaM 34B
- **Built by**: 120+ specialists including 35 PhD researchers; refined by 600 domain experts

- **2026 Plans**: Data center expansions in Riyadh/Dammam; $10B VC fund for Arabic AI

## 17. Saudi National Address API (Saudi Post / SPL)

### National Address API
- **URL**: https://api.address.gov.sa/
- **SPL Info**: https://splonline.com.sa/en/national-address-api/
- **What it does**: Geographic/spatial data for Saudi addresses -- search, validation, geocoding
- **API Endpoints**:
  - **Free Text Search** -- Search addresses with free text
  - **Fixed Search** -- Structured address search
  - **Nearest POIs** -- Find nearest points of interest
  - Address validation, city lists, neighborhood data
- **Pricing**: Free registration; subscription tiers available (details on portal)
- **Access**: Register online for free; works across all operating systems
- **Mandatory**: As of January 1, 2026, National Address usage is mandatory in Saudi Arabia
- **Libraries**: PHP/Laravel wrapper available (github.com/naifalshaye/saudiaddress)

## BONUS: Additional Key Saudi APIs Discovered

### Wathq (Digital Verification Platform)
- **URL**: https://developer.wathq.sa/en
- **What it does**: Real-time verification of government data through API
- **APIs**:
  - Commercial Registration (Ministry of Commerce) -- 2-12 SAR/call
  - Commercial Contract Issuing -- NEW
  - Power of Attorney (Ministry of Justice)
  - Real Estate Deeds (Ministry of Justice)
  - National Address (Saudi Post)
  - Employee Information (Masdr/GOSI) -- 7 SAR/call
- **Pricing**:
  - **Trial**: Free, 100 inquiries, 30 days
  - **PrePaid**: From 5,000 SAR (30 days) to 150,000 SAR+12% bonus (1 year)
  - **Enterprise**: Custom pricing, 50-100 req/sec, unlimited inquiries
- **Access**: Register online; sandbox available for testing

### Najiz (Ministry of Justice)
- **URL**: https://developers.najiz.sa/en
- **API Catalog**: https://developers.najiz.sa/en/api-catalog
- **What it does**: Judicial and notarization services integration
- **Access**: Developer registration required

### Etimad (Government Procurement)
- **URL**: https://apiportal.etimad.sa/
- **What it does**: Government contract data, salary certificates, procurement integration
- **APIs**: Contract data inquiry, salary certificates, discounts, payroll, bank contracts
- **Access**: Register on the developer portal

### ZATCA (Zakat, Tax, Customs)
- **URL**: https://zatca.gov.sa/ar/PublicData/Pages/APIs.aspx
- **What it does**: E-invoicing (Fatoora) integration
- **Key API**: Fatoora Platform API -- real-time e-invoice clearance
- **Requirements**: XML with embedded PDF/A-3, UUID, digital signature, cryptographic stamp
- **Mandatory**: Phase 2 integration waves ongoing through June 2026

### DGA API Inventory
- **URL**: https://dga.gov.sa/en/listing-services/API-Inventory
- **What it does**: Central registry of government APIs for cross-entity integration

### Nafath (Digital Identity)
- **What it does**: National digital identity authentication (SSO for government/private services)
- **Integration**: Via Technology Control Company (TCC) license; uses biometric/NFC on smartphones
- **Libraries**: Laravel package available (github.com/mohamad-zatar/saudi-nafath-integration)
- **Access**: Requires license from TCC

### Elm Services (Yakeen, Muqeem)
- **URL**: https://www.elm.sa/
- **Yakeen API**: Electronic authentication/identity verification via NIC (launched 2004). SOAP-based
- **Muqeem API**: Residency/visa management system (launched 2007)
- **Access**: Enterprise/government integration only; not publicly self-serve

### Saudi Open Data Portal
- **URL**: https://data.gov.sa/en
- **Developer Endpoint**: https://data.gov.sa/en/developers
- **National Data Lake**: https://lake.data.gov.sa/
- **Stats**: 11,439+ datasets from 289 organizations; 284,800+ downloads
- **Formats**: CSV, JSON, XML; many datasets accessible via API
- **Access**: Free; no registration required for download

### Salla (Saudi eCommerce Platform)
- **URL**: https://salla.dev/ | https://docs.salla.dev/
- **What it does**: eCommerce API for 68,000+ merchants in Saudi/GCC
- **APIs**: RESTful Merchant APIs -- products, orders, customers, payments, shipping, themes
